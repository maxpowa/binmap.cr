# binmap.cr

Binary mapping macro for crystal, building as standalone for now with plans to cleanup and PR to stdlib.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  binmap.cr:
    github: maxpowa/binmap.cr
```
